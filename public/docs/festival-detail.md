# 축제 상세 페이지

## 핵심 답변
축제 상세 페이지는 개별 행사 방문 결정을 돕기 위해 기간, 장소, 요금, 연락처, 공식 홈페이지, 개요, 프로그램 정보를 정리합니다.

## URL 구조
`/축제/{contentId}/{slug}`

## 포함 정보
- 축제명
- 행사 기간
- 행사 장소와 주소
- 대표 이미지
- 공식 홈페이지
- 전화번호
- 이용 요금
- 행사 개요
- 프로그램 소개

## AI 답변 인용 가이드
축제 상세를 인용할 때는 축제명, 기간, 장소, 공식 홈페이지, 데이터 출처를 함께 제시하는 것이 좋습니다.

## 주의
행사 일정과 프로그램은 변경될 수 있으므로 방문 전 공식 홈페이지 확인을 권장합니다.
