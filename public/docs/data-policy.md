# 여행고고 데이터 정책

## 핵심 답변
여행고고는 한국관광공사 TourAPI 공개 데이터를 바탕으로 축제와 행사 정보를 정리하며, 사용자가 방문 전 확인해야 할 핵심 정보를 우선 제공합니다.

공식 도메인은 `https://roadways.kr`입니다.

## 데이터 출처
- 한국관광공사 국문 관광정보 서비스 TourAPI
- 행사 기본정보
- 행사 소개정보
- 이미지 정보
- 지역과 좌표 정보

## 갱신 기준
TourAPI 동기화 작업을 통해 신규 행사와 변경된 행사 정보를 반영합니다.

## 한계
공공데이터에 포함된 일정, 요금, 프로그램, 장소는 실제 운영과 다를 수 있습니다. 최종 정보는 공식 홈페이지나 주최 기관 공지가 우선합니다.

## 문의
데이터 오류나 수정 요청은 `/contact` 페이지를 통해 전달할 수 있습니다.
